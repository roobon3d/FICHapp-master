FichApp es una aplicación para dispositivos con Sistema Operativo Android, creada con
el objetivo de registrar las horas laborales de un empleado dentro de una empresa, en
la cual actuará como administrador. Para ello se debe seguir los siguientes pasos:

1) Una vez descargada e instalada la aplicación en el dispositivo, se debe dar de alta el usuario Administrador(responsable en la empresa), rellenando los campos del formulario inicial, id(identificador del responsable), contraseña que va a usar.

Inicio sesión como Administrador/Gestor:

2) El Administrador rellena los datos de la empresa en los campos correspondientes.

3) El Administrador inicia sesión rellenando id y contraseña y en la siguiente pantalla
elige su rol: Gestor.

4) El Gestor accede a Menú Gestor que tiene las siguientes opciones:
- Editar Datos (Gestor)
- Editar Datos (Empresa)
- Registro de Empleados: El Gestor registra a los empleados, rellenando datos personales(del empleado), le asigna un id(empleado) y una contraseña inicial.
- Consulta de Datos Empleado
- Enviar Informe

Inicio sesión como Empleado:

1) El Empleado rellena los campos: id(Usuario) y contraseña.
2) Accede a Menú Empleado que tiene las opciones:
- Registro horas Empleado
- Consulta de horas
